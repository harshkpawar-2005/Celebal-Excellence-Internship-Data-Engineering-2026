# SQL Assignment - Sales Data Analysis

## Overview

This assignment was completed as part of the **Week 2** module of the **Celebal Technologies Data Engineering Internship**.

The objective was to design a relational database and perform SQL-based data analysis using MySQL. The assignment covers database creation, filtering, aggregation, joins, transactions, and other core SQL concepts.

---

## Assignment Objectives

- Design a relational database.
- Create tables with appropriate constraints.
- Implement primary and foreign key relationships.
- Create indexes for query optimization.
- Load sample data into the database.
- Perform SQL-based business analysis using filtering, aggregation, joins, and transactions.

---

## Database Schema

The database consists of four related tables:

- Customers
- Products
- Orders
- Order Items

These tables are connected using primary keys and foreign keys to maintain referential integrity.
