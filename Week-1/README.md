## Assignment Completed

### Shopping Dataset Analysis

#### Objective
Perform exploratory data analysis (EDA), data cleaning, feature engineering, and visualization on a shopping dataset using Pandas.

#### Tasks Performed
- Loaded the dataset into a Pandas DataFrame
- Explored dataset structure and statistics
- Identified and handled missing values
- Checked and removed duplicate records
- Converted price-related columns into numeric format
- Created derived features:
  - `price_difference`
  - `popularity_score`
- Performed category-wise and pricing analysis
- Created visualizations
- Saved the cleaned dataset
